# PRODIGY_ML_02

## Customer Segmentation using K-Means Clustering

### Objective

Create a K-Means clustering algorithm to group customers of a retail store based on their purchase history.

### Tools Used

* Python
* Google Colab
* Pandas
* Scikit-Learn
* Matplotlib

### Dataset

Mall Customers Dataset

### Features Used

* Annual Income (k$)
* Spending Score (1-100)

### Workflow

1. Loaded the customer dataset.
2. Selected Annual Income and Spending Score as features.
3. Used the Elbow Method to determine the optimal number of clusters.
4. Applied K-Means Clustering.
5. Segmented customers into different groups.
6. Visualized customer clusters and cluster centers.

### Results

* Dataset Shape: (200, 5)
* Number of Clusters: 5

### Outcome

Successfully grouped customers into distinct segments based on spending behavior and annual income using K-Means Clustering. This helps businesses understand customer patterns and improve marketing strategies.
